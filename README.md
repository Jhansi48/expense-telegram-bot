🎙️ Voice-Based Expense Tracking Bot

A Telegram bot that converts voice messages into structured expense records using OpenAI Whisper. The bot automatically categorizes expenses and stores them in an Excel file, making personal expense tracking fast and effortless.

✨ Features:

🎤 Accepts voice messages through Telegram
📝 Converts speech to text using OpenAI Whisper
🏷️ Automatically categorizes expenses
📊 Stores expense records in Excel
📅 Maintains date and time of each transaction
⚡ Fast and easy expense logging

🛠️ Tech Stack:

Python
OpenAI Whisper
Telegram Bot API
Pandas
OpenPyXL
Regex
FFmpeg
